== Installation ==

1. Download the plugin ZIP file.
2. In your WordPress dashboard, go to **Plugins > Add New > Upload Plugin**.
3. Upload `imjolwp-ai-automation.zip`, then click **Install Now**.
4. Click **Activate Plugin**.
5. Go to **ImjolWP AI > Settings** to configure your DeepInfra API Key.

== Usage ==

1. Go to **ImjolWP AI > AI Post Generator**.
2. Fill in:
    - Post Title  
    - Focus Keywords  
    - Minimum Word Count  
    - Language  
    - Post Status (Draft/Publish)  
    - Post Type (Post/Page)  
3. Click **Generate Content**.
4. Optional: Enable **Schedule Automation** to generate content at a later time.
