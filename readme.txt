== Installation ==

1. Download `imjolwp-ai-automation.zip` from WordPress.org.
2. Log in to your WordPress dashboard.
3. Navigate to Plugins > Add New.
4. Click on Upload Plugin and select the plugin zip file.
5. Click Install Now, then Activate the plugin.

== Frequently Asked Questions ==

= How do I change the language of AI-generated content? =
You can select your preferred language when creating a new post.

= Does the plugin support custom post types? =
Currently, the plugin supports standard WordPress posts. Custom post type support is planned for future updates.

= Can I disable AI content generation? =
Yes, you can disable specific AI features in the ImjolWP Settings page.
